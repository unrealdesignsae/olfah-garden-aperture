# Olfah — Garden Aperture

A cinematic, responsive landing-page concept for Alef Group's Olfah community in Sharjah.

## Highlights

- Scroll-led architectural storytelling with GSAP and Lenis
- Responsive desktop and mobile layouts
- Official Olfah project imagery, project statistics, amenities, gallery, map, and brochure
- Exterior/interior gallery with lightbox navigation
- Accessible navigation, reduced-motion support, and validated enquiry states
- Vite, React, Tailwind CSS, GSAP, and Lenis

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

Project facts and media were sourced from the [official Olfah project page](https://www.alefgroup.ae/alef-communities/olfah/).
